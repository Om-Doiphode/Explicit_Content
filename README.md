
# Fake News and Explicit Content Detection 

The Fake News and Explicit Content Detection project is aimed at developing a system that can identify and flag fake news articles and explicit content in various forms of media. This project leverages advanced natural language processing (NLP) and computer vision techniques to analyze and classify content, contributing to a more trustworthy online information environment.




## Features

- Fake News Detection: Utilizes NLP algorithms to analyze text content and identify potential fake news articles.
- Explicit Content Detection:Incorporates computer vision algorithms to analyze images and videos, flagging explicit or inappropriate content.
- User-Friendly Interface: Provides a user-friendly interface for users to interact with the detection system easily

## Installation
`git clone git@github.com:Om-Doiphode/Explicit_Content.git`

# Usage

- Input Content: Provide link for analysis through the user interface.
- Analyze: Click the "Submit" button to initiate the detection process.
- View Results: The system will display results indicating whether the content is flagged as fake news or explicit.

## Frontend Setup
### Navigate to the frontend directory
`cd WebApp/frontend`

### Install dependencies
`npm install`

### Start the frontend development server
`npm start`

## Backend Setup
### Open another terminal
`cd WebApp/backend`

### Install backend dependencies
`npm install`

### Start the backend server using nodemon
`nodemon server.js`
